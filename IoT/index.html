How It Works


Smart Fan Speed Controller Based on Room Temperature

This IoT-based fan speed controller automatically adjusts the fan speed based on real-time room temperature. Using an ESP8266 (NodeMCU) microcontroller, the system reads temperature data from a DHT11 sensor. Based on this data, the fan speed is regulated using a MOSFET to control the power supplied to a 12V DC fan. The temperature and fan status are displayed on a 1.3' OLED display for easy monitoring.

Key Features:
Remote Control: Fan speed and temperature range settings can be adjusted through an app, allowing convenient control from anywhere. This intelligent system ensures that your room stays comfortable, with the fan working only as needed for efficient cooling.

Automatic Speed Adjustment: Fan speed changes based on the measured room temperature.

Real-time Temperature Monitoring: Temperature is continuously displayed on the LCD screen.

Efficient Power Usage: The fan operates at the required speed, optimizing energy consumption.

Components Used



  • Node MCU ESP8622 Module
  • DS18B20 Temperature Sensor
  • 16x2 LCD Display ,Breadboard
  • 10k Potentiometer , Resistor 10k and 4.7k
  • IRF540N Mofset , 7805 Voltage Regulator IC
  • 12v DC Fan , 100uf & 10uf Capacitor
  • 12v DC Power Adapter , Connecting Wires
